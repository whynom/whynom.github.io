Doing my website in Org mode

Org mode is one of those emacs tools that makes people want to start using emacs. It certainly was for me.

I would be lying if I didn’t say that I suffer from software tool paralysis. I want to use the right tool to get the job done from the get go, but I constantly find myself wandering around to various software pacakages that have some extraordinarily neat features, but usually mean me starting from scratch as far as how to use that tool effectively.

This has led me to decide to use org-mode to do all of my writing. I’m going to be doing my writing in org-mode at some point in my writing workflow and I might as well just export directly from that file, right?

Also, even if I do decide to use something like Hugo for org mode in the future, I’ll more or less have a file that can be exported to that kind of webpage as well.

For the record, I’m using this for exporting to html in emacs with some bootstrap prettiness
